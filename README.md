# elmr

**Ham radio software, rebuilt for everywhere.**

A lot of great amateur-radio software got stranded — written years ago, for one
OS, hard to install and harder to contribute to. elmr rebuilds the everyday tools
— decoding, logging, tracking, learning — to run anywhere, install in seconds, and
improve in the open.

Apps come first and earn their keep. Shared packages get pulled out, named, and
published underneath them once a second project needs them — not before.

("Elmer" is the ham term for a mentor, the operator who helps a newcomer get
licensed and on the air. That's the idea.)

## On the air

- **[morse.elmr.dev](https://morse.elmr.dev)** — browser CW decoder. Pulls Morse

## Building

Repos here run on [Bun](https://bun.sh) (`bun.lock` is the source of truth in CI)
and ship through GitHub Actions. Clone, `bun install`, and the README in each repo
takes it from there.

Contributions welcome. The door to the shack is open — issues, PRs, and questions
all land in [Discord](https://discord.elmr.dev) or the relevant repo.
